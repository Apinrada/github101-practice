## Reflection
วันนี้ฉันเรียนรู้อะไรจากการใช้ GitHub: วันนี้ฉันได้เรียนรู้ว่า GitHub ไม่ได้ใช้เฉพาะสำหรับเก็บ Source Code เท่านั้น แต่ยังสามารถใช้เป็นพื้นที่กลางสำหรับจัดเก็บเอกสาร หลักฐานการทำงาน และติดตาม Contribution ของสมาชิกในทีมได้
ฉันได้เรียนรู้การสร้างไฟล์ใหม่ การแก้ไขไฟล์ การใช้งาน Markdown และการจัดระเบียบข้อมูลภายใน Repository ให้ตรวจสอบได้ง่าย

ฉันติดปัญหาอะไร: ในช่วงแรกฉันยังไม่คุ้นเคยกับโครงสร้างของ Repository และยังไม่แน่ใจว่าควรจัดเก็บหลักฐานการทำงานไว้ในตำแหน่งใดจึงจะเหมาะสม

ฉันแก้ปัญหาอย่างไร: ฉันศึกษาคำแนะนำจาก README ของ Repository และทดลองใช้งาน GitHub Web ด้วยตนเองจนสามารถสร้างไฟล์ แก้ไขข้อมูล และจัดเก็บหลักฐานการทำงานได้อย่างถูกต้องตามรูปแบบที่กำหนด
